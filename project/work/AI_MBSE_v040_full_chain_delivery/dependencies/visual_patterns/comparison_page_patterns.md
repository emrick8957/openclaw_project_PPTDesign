# 对比页模式

## 1. 左右对比

左侧对象 A，右侧对象 B，中间差异和结论。

适用：竞品对比、现状与目标对比、方案对比。

## 2. 六维矩阵

行：维度。列：对象 A / 对象 B / 判断 / 建议。

适用：体系化对比。

## 3. 差距热力图

行：差距项。列：影响、紧急度、补齐难度、动作。

适用：差距清单、风险识别。

## 4. 表格证据区 + 指标结论卡

结构：左/中部为弱边框对比表，右侧为 2~3 个关键指标卡，底部为弱结论区。

适用：模型评测结果、竞品指标对比、方案效果对比、基准测试解读。

使用规则：

- 表格承担证据，不承担全部结论；
- 右侧指标卡承担“领导一眼看到的判断”；
- 表格行列线使用浅灰细线，高亮项不超过 2~3 个；
- 指标卡统一为“指标名—数值—一句解释”；
- 底部结论条使用窄条或灰底红字，不使用厚重红色横幅。
