# 高层总览页模式

## 1. 一页总览四模块

模块：总体判断 / 关键差距 / 主要风险 / 建议动作。

适用：P2 核心结论页。

## 2. 关键数字 + 能力地图

模块：顶部核心判断，左侧关键数字，右侧能力图谱。

适用：战略价值、业务规模、建设成效。

## 3. 结论区 + 案例区 + 决策区

模块：左侧结论，中部案例或图表，右侧决策建议。

适用：面向领导的 1 分钟速读页。

## 4. 左主判断 + 右三卡 + 轻流程

模块：左侧一句主判断 + 3 个短要点；右侧 3 张关键发现卡片；底部使用细线流程或弱结论区。

适用：研究报告、技术洞察、评测解读的首页或核心判断页。

使用规则：

- 左侧判断不超过 3 个短要点，避免长段落；
- 右侧三卡必须等宽/等高/同内边距；
- 强卡片最多 1 张，其余卡片降为中/弱视觉；
- 底部流程只使用细线箭头或灰底弱说明，不使用厚重红色压底；
- 红色只用于核心数字、关键词或主判断中的 1 个锚点。
