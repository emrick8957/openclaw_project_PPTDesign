# 风险与决策页模式

## 1. 风险矩阵

维度：风险项 / 概率 / 影响 / 预警信号 / 应对措施 / 责任方。

## 2. 决策清单

维度：决策事项 / 推荐方案 / 资源需求 / 时间节点 / 预期产出。

## 3. 闭环治理图

发现 → 分析 → 处置 → 复盘 → 机制优化。

## 4. 问题-动作-责任表

维度：问题 / 影响 / 根因 / 动作 / 责任 / 截止时间。

## 使用规则

风险和决策页必须能回答：

- 需要谁拍板；
- 拍什么板；
- 不拍板的影响；
- 需要多少资源；
- 何时看到结果。

## 5. 三类失败模式诊断页

结构：三张问题卡片 + 底部弱结论区。

适用：失败模式、风险类型、能力短板、问题归因。

使用规则：

- 三张卡片可以同构，但必须通过编号、关键词、图标或强弱层级建立扫读路径；
- 至多突出 1 张强卡片，其余使用中/弱视觉；
- 每张卡片建议采用“关键词 + 表现 / 风险 / 改进方向”结构；
- 避免长句堆叠，优先短语化；
- 底部结论区必须弱化，不得使用厚重红色压底条。
