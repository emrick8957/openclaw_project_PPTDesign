# 华为风格参考摘要

## v0.1 模板提取

来自用户提供的华为模板：

- 封面页包含部门、作者、日期、Security Level；
- 目录页简洁；
- 品牌色包含 PANTONE 186C、PANTONE 185C、红/橙/灰/黑/白体系；
- 图表配色统一；
- 页脚包含 Huawei Confidential 与页码；
- Thank you 页包含版权、免责声明、愿景句。

## v0.2 版式截图提取

来自用户提供的 27 张版式截图：

- 高密度但有秩序；
- 顶部结论标题 + 中部结构图表 + 底部页脚；
- 多使用左右对比、三栏卡片、四象限、分层架构、时间轴、生态地图、风险矩阵；
- 红色用于结论、路径箭头、关键数字；
- 灰色用于底图、分区、辅助文字；
- 复杂页面常配“一句话判断”。

## v0.3.3 SWE Atlas 页面图片对比增量

来自 SWE Atlas 页面图片与 `tmp/PPTTemplate/0517` 华为风格模板对比，新增以下稳定视觉判断：

- 华为风格不是“红色越多越像”，而是红色只做少量视觉锚点；正文页主红色焦点应控制在 1-2 个。
- 技术研究报告类页面容易偏重，应通过弱边框、浅灰线、窄结论条和稳定页脚降噪。
- 表格页应采用“表格证据化 + 指标结论化”：表格弱化，关键判断放在标题或右侧指标卡。
- 流程/泳道/闭环图必须统一节点、箭头、连接线，避免图形元素混搭。
- 卡片页必须建立强/中/弱层级，避免三卡或多卡全部同等强调。
- 页脚是正式感来源之一，必须稳定但不抢主体。
