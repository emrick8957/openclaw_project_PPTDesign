# 版式截图分析提示词

请分析用户提供的 PPT 版式截图，并抽象为可复用规律。

输出：
1. 页面类型识别；
2. 页面在整套汇报中的角色；
3. 版式结构；
4. 色彩和字体规律；
5. 图表模式；
6. 信息密度；
7. 标题句式与结论表达方式；
8. 论证结构与页面收口方式；
9. 可复用场景；
10. 不应复用的风险；
11. 建议沉淀到 `visual_patterns/` 的规则；
12. 建议沉淀到 `methodology_patterns/` 的规则；
13. 哪些内容只能作为主题知识，不得上升为通用规则。

注意：
- 只沉淀可复用规律，不把截图中的具体主题结论写成全局知识；
- 截图中的数据、案例、结论默认不直接作为事实依据；
- 若同类规律只在单页中出现，优先标记为“局部参考”，不要直接升格。
