## 4. 新增 `skills/huawei_ppt_master/core/reference_ingestion_workflow.md`

### 4.1 建议正文草案
```markdown
# 参考材料学习流程

## 1. 目标
当用户持续提供 PPT、PPT 图片、模板、文本材料或数据材料时，本流程负责：
1. 对参考材料进行分类；
2. 从视觉与内容两个通道抽取可复用规律；
3. 判断规律应沉淀到哪一层；
4. 控制过拟合、主题污染与事实误用风险；
5. 让后续生成稳定复用已沉淀规则。

## 2. 输入类型分类
常见输入包括：
- 完整 PPT / PPTX；
- PDF 导出的多页截图；
- 单页页面截图；
- 模板页 / 样张页；
- 文本材料；
- 数据表；
- 行业研究或内部分析材料。

## 3. 证据等级
每份材料必须先判断证据等级：
1. 风格参考：用于学习配色、密度、视觉气质；
2. 结构参考：用于学习页面布局、章节结构、图表组织；
3. 论证参考：用于学习标题句式、页型分工、论证套路、决策收口；
4. 事实依据：用于支持具体数据、案例、参数或市场判断。

同一材料可以同时具备多个等级，但事实依据要求最高，不能由截图自动替代。

## 4. 双通道抽取
### 4.1 视觉通道
抽取：
- 配色规律；
- 字体层级；
- 页面密度；
- 页脚规范；
- 常见版式结构；
- 图表主形态；
- 强调信息方式。

### 4.2 内容通道
抽取：
- 章节逻辑；
- 页型角色；
- 标题句式；
- 一页如何形成核心判断；
- 论证链条；
- 决策收口方式；
- 行业框架与方法论套路。

## 5. 沉淀落位规则
抽取结果必须按层级落位：
1. 通用视觉规律 → `templates/`、`visual_patterns/`
2. 通用表达规律 → `templates/wording_rules.md`、`templates/narrative_patterns.md`
3. 通用方法论规律 → `methodology_patterns/`
4. 主题专属框架 → `domain_profiles/`
5. 一次性案例 → 只作为当前任务参考，不进入长期规则库

## 6. 升级条件
只有满足以下条件，才允许进入长期规则：
1. 跨两份及以上材料重复出现；
2. 对多个主题或多个任务有复用价值；
3. 不与现有通用规则冲突；
4. 不依赖单一客户、单一项目、单一行业私有上下文。

## 7. 冲突处理
如果新材料与已有规则冲突：
1. 优先保守，不直接覆盖旧规则；
2. 先判断是否属于主题专属差异；
3. 再判断是否只是单次案例；
4. 必要时将冲突规则降级到主题层或当前任务层。

## 8. 复用优先级
生成时按以下优先级复用：
1. 已验证的通用规则；
2. 当前主题命中的主题规则；
3. 已验证的方法论规则；
4. 当前任务一次性参考；
5. 用户当前明确指定偏好。

## 9. 禁止事项
- 不得把截图中的具体市场结论直接当作事实；
- 不得把单份材料的专有术语升格为通用规则；
- 不得把行业专属章节逻辑写成所有主题的默认结构；
- 不得因参考材料学习而放松主题污染控制。
```

## 5. 新增 `skills/huawei_ppt_master/core/reference_material_policy.md`

### 5.1 建议正文草案
```markdown
# 参考材料使用边界

## 1. 目的
定义不同参考材料的证据等级、可用范围与禁止用途，避免把视觉参考误当事实依据。

## 2. 材料类型与默认用途
### 2.1 图片 / 截图
默认可用于：
- 风格参考；
- 结构参考；
- 页面角色识别；
- 图表与版式抽象。

默认不可直接用于：
- 事实断言；
- 精确数据引用；
- 未核验案例引用。

### 2.2 PPT / PPTX 正文
可用于：
- 表达习惯参考；
- 章节结构参考；
- 论证套路参考；
- 页面组织参考。

涉及具体事实时：
- 需要核验其来源是否仍有效；
- 未核验时必须继续标注“待补材料”或“需确认”。

### 2.3 文本材料 / 行业报告
可用于：
- 论证参考；
- 章节结构参考；
- 事实依据候选。

若为权威来源且时效可接受，可作为事实依据。

### 2.4 数据表
可优先作为：
- 图表数据来源；
- 对比和趋势依据；
- 定量支撑。

但仍需检查：
- 口径是否一致；
- 时间是否有效；
- 是否适合直接外显。

## 3. 使用边界
1. 风格参考不等于事实依据；
2. 结构参考不等于通用方法论；
3. 论证参考不等于可直接复述原文；
4. 事实依据必须优先选择可核验来源。

## 4. 输出要求
如果输出中使用了未完全核验的材料，只能做方向性表达，不得冒充确定事实。
```

```markdown
# 参考材料学习专项回归

## 1. 目标
验证 Skill 在持续吸收 PPT / PPT 图片后，是否真正提升了样式、结构、表达和体系层能力，同时没有引入主题污染或过拟合。

## 2. 回归项
### 2.1 视觉学习回归
检查是否学到配色、密度、标题区、页脚区、图表主形态。

### 2.2 结构学习回归
检查是否学到章节编排、页型分工、页面结构，而非只学到表面措辞。

### 2.3 表达学习回归
检查标题是否更结论化，收口是否更像高层汇报。

### 2.4 体系学习回归
检查是否能从多套材料中抽取稳定论证框架与行业套路。

### 2.5 主题隔离回归
检查是否因参考材料带入无关术语或错误主题框架。

### 2.6 增量稳定性回归
检查新增材料是否无依据破坏已有通用规则。

## 3. 一票否决项
- 非 AI 主题被 AI 算力术语污染；
- 截图中的具体结论被直接当成事实；
- 单客户材料写成全局规则；
- 新规则与旧规则冲突但未降级处理。
```
