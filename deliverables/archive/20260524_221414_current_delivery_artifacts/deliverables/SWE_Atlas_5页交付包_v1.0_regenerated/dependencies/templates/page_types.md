# 通用页型库

## 1. 封面页 cover

适用：所有 PPT。

结构：主标题 + 副标题 + 汇报对象/部门/作者/日期/Security Level。

## 2. 目录页 agenda

适用：超过 8 页的 PPT。

结构：3~5 个章节，不超过 7 个一级条目。

## 3. 核心结论页 executive_summary

适用：高层汇报、战略汇报、调研报告。

结构：核心判断 + 3~5 个关键结论 + 决策建议。

## 4. 背景趋势页 context_trend

结构：外部趋势 + 内部诉求 + 核心矛盾。

## 5. 问题诊断页 problem_diagnosis

结构：现状问题 + 影响 + 根因 + 解决抓手。

## 6. 对比分析页 comparison

结构：对象 A / 对象 B / 判断 / 启示。

## 7. 总体架构页 architecture

结构：资源层 / 平台层 / 能力层 / 应用层 / 运营层。

## 8. 能力地图页 capability_map

结构：能力域 + 关键能力 + 成熟度 + 补齐动作。

## 9. 路线图页 roadmap

结构：短期 / 中期 / 长期，或试点 / 沉淀 / 复制。

## 10. 风险闭环页 risk_matrix

结构：风险 / 影响 / 概率 / 应对措施 / 责任主体。

## 11. 决策建议页 decision_table

结构：决策事项 / 推荐方案 / 资源需求 / 时间节点 / 预期产出。

## 12. Thank you 页 thank_you

结构：Thank you + 版权/免责声明/愿景句。
