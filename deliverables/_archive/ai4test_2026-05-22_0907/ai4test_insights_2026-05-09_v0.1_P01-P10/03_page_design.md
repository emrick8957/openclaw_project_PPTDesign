# AI4Test洞察 - 页面设计说明 v0.1
## P1
页面类型：cover
版式模式：executive_summary_dashboard
区域划分：顶部标题区 + 中部副标题区 + 底部信息区。
标题区：大标题左对齐，突出“AI4Test洞察”。
图表区：无主图表。
文字区：副标题写“价值判断、推进路径与拍板建议”。
强调元素：标题中的 AI4Test 用华为红强调。
页脚：Huawei Confidential + 页码。
配色建议：白底，红黑灰主色。
信息密度控制：封面不超过 3 个信息块。
PPTX Builder 注意事项：不要堆额外装饰元素，保持正式汇报气质。
可能的压缩/拆页建议：无。

## P2
页面类型：executive_summary
版式模式：executive_summary_dashboard
区域划分：顶部标题区 + 中部四卡片区 + 右侧决策提示区 + 底部收口区。
标题区：结论型标题单行或双行呈现。
图表区：关键结论卡片组作为主图表。
文字区：每张卡片 1 句 headline + 1 句解释。
强调元素：右侧“决策提示”卡片标红边。
页脚：Huawei Confidential + 页码。
配色建议：卡片白底浅灰描边，关键字用红色。
信息密度控制：卡片总数控制在 4 张。
PPTX Builder 注意事项：卡片高度统一，右侧提示区宽度不超过页面 25%。
可能的压缩/拆页建议：若内容超载，优先压缩解释句，不新增卡片。

## P3
页面类型：executive_summary
版式模式：four_quadrant_judgement
区域划分：顶部标题区 + 中部四象限区 + 底部原则区。
标题区：结论型标题居左。
图表区：优先级矩阵为主，四类场景分布在象限中。
文字区：底部 3 条先行选择原则。
强调元素：高价值低难度象限用淡红底突出。
页脚：Huawei Confidential + 页码。
配色建议：红色强调优先区，灰色呈现次级区。
信息密度控制：象限内每项不超过 1 行。
PPTX Builder 注意事项：保持坐标轴清晰，避免象限文字拥挤重叠。
可能的压缩/拆页建议：如场景项过多，只保留一级场景名称。

## P4
页面类型：problem_diagnosis
版式模式：insight_panel_with_chart
区域划分：左侧主表区 + 右侧洞察区 + 底部结论条。
标题区：顶部统一标题栏。
图表区：对比表呈现传统模式与新矛盾。
文字区：右侧列 3 条影响判断。
强调元素：底部红色结论条写核心矛盾。
页脚：Huawei Confidential + 页码。
配色建议：主表黑灰为主，核心影响用红点标识。
信息密度控制：表格控制在 4 行以内。
PPTX Builder 注意事项：右侧洞察与主表高度对齐，底部结论条独立悬浮。
可能的压缩/拆页建议：无。

## P5
页面类型：context_trend
版式模式：trend_curve_with_strategy
区域划分：左侧趋势曲线区 + 右侧策略含义区。
标题区：顶部统一标题栏。
图表区：趋势曲线展示三阶段演进。
文字区：右侧 3 条组织含义。
强调元素：当前阶段节点用红色圆点突出。
页脚：Huawei Confidential + 页码。
配色建议：曲线用深灰，关键节点与结论用红色。
信息密度控制：右侧不超过 4 条要点。
PPTX Builder 注意事项：曲线需清晰标注阶段名称，右侧文字不要压缩过小。
可能的压缩/拆页建议：无。

## P6
页面类型：risk_matrix
版式模式：risk_decision_matrix
区域划分：上方标题区 + 中部热力图区 + 右上判断卡片区。
标题区：结论型标题。
图表区：差距热力图按影响、紧急度、难度排列。
文字区：右上角补“窗口期判断”。
强调元素：高影响高紧急项标红。
页脚：Huawei Confidential + 页码。
配色建议：热力等级使用灰到红渐变，但整体保持克制。
信息密度控制：风险项控制在 4 项。
PPTX Builder 注意事项：避免使用高饱和大面积色块，保留商务感。
可能的压缩/拆页建议：如风险项增加，拆成“损失”与“难度”两页。

## P7
页面类型：capability_map
版式模式：executive_summary_dashboard
区域划分：顶部标题区 + 中部四场景卡片区 + 右侧适配前提区。
标题区：结论型标题。
图表区：能力地图，以 4 个场景卡片为主。
文字区：右侧解释适配条件与边界。
强调元素：优先场景加红色标签“第一阶段”。
页脚：Huawei Confidential + 页码。
配色建议：卡片用浅灰底，一级标题红色。
信息密度控制：每卡片 3 个字段以内。
PPTX Builder 注意事项：卡片间距均匀，右侧说明区不要挤压卡片宽度。
可能的压缩/拆页建议：如要展开案例，后续另拆场景页。

## P8
页面类型：architecture
版式模式：layered_architecture
区域划分：顶部标题区 + 中部四层架构区 + 右下缺口提示区。
标题区：结论型标题。
图表区：分层堆栈图展示数据、规范、工具链、责任边界。
文字区：右下角补当前常见缺口。
强调元素：治理层与责任边界层用红色描边。
页脚：Huawei Confidential + 页码。
配色建议：分层用灰白区分，关键层红色线框强调。
信息密度控制：每层不超过 4 个子项。
PPTX Builder 注意事项：层级关系要明显，自下而上逻辑清晰。
可能的压缩/拆页建议：如要展开工具链接口，可另拆一页。

## P9
页面类型：comparison
版式模式：two_column_comparison
区域划分：左栏增强场景 + 右栏谨慎替代场景 + 中部判断箭头 + 底部原则。
标题区：顶部标题栏。
图表区：双栏对比表。
文字区：底部 3 条判断原则。
强调元素：中部箭头写“人机协同边界”。
页脚：Huawei Confidential + 页码。
配色建议：左栏浅红强调“优先增强”，右栏浅灰强调“谨慎替代”。
信息密度控制：每栏 4 项以内。
PPTX Builder 注意事项：左右栏宽度保持一致，中部箭头区窄而清晰。
可能的压缩/拆页建议：无。

## P10
页面类型：risk_matrix
版式模式：risk_decision_matrix
区域划分：标题区 + 主风险表区 + 底部收口区。
标题区：结论型标题。
图表区：风险矩阵表。
文字区：底部 1 行总结“治理先于扩张”。
强调元素：P0 风险行标红。
页脚：Huawei Confidential + 页码。
配色建议：高风险项红色强调，其余灰黑色。
信息密度控制：风险项控制 4~5 行。
PPTX Builder 注意事项：表头固定，列宽优先保证“应对措施”可读性。
可能的压缩/拆页建议：如要展开责任机制，可拆为治理专页。
