# 高层论证套路

## 1. 目标
沉淀高层汇报中反复出现的“判断—依据—动作—风险—决策”套路，用于提升 PPT 的战略表达、决策收口与可执行性。

## 2. 常见论证骨架
### 2.1 判断 → 拆解 → 路径 → 拍板
适用于战略规划、竞争分析、资源申请。

### 2.2 趋势 → 冲击 → 机会 → 动作
适用于产业趋势、市场变化、技术演进类汇报。

### 2.3 差距 → 根因 → 抓手 → 验收
适用于问题诊断、能力提升、运营治理、项目复盘。

### 2.4 现状 → 风险 → 选择 → 决策
适用于风险提示、关键选择、管理层拍板页。

## 3. 适用边界
- 可作为通用高层汇报方法论；
- 不得把某一行业或某一客户的具体结论写成通用套路；
- 使用时必须结合当前主题、汇报对象和材料证据强度。
