# cybersecurity 网络安全主题包

## 使用时机

主题涉及网络安全、安全运营、SOC、主机安全、系统加固、漏洞、威胁、合规、等保等。

## 推荐结构

安全形势 → 资产与风险 → 现状问题 → 能力框架 → 监测处置 → 合规治理 → 运营机制 → 建设路线 → 决策事项。

## 推荐表达

- 安全建设重点应从单点工具采购转向持续监测、快速处置和闭环运营。
- 关键能力包括资产识别、风险发现、威胁检测、事件响应、合规审计和持续加固。
- 高层应关注风险可视、责任清晰、处置闭环和合规可审计。
