# project_operation 持续运营主题包

## 使用时机

主题涉及持续运营、运维、运营规划、年度目标、SLA、运营指标等。

## 推荐结构

运营目标 → 运营对象 → 当前问题 → 机制设计 → 指标体系 → 闭环流程 → 风险处置 → 阶段计划 → 资源保障。

## 推荐表达

- 持续运营的关键不是项目交付完成，而是能力稳定运行、问题可闭环、价值可度量。
- 应建立目标、指标、机制、责任和复盘的完整运营闭环。
- 建议用月度指标看过程、季度复盘看效果、年度目标看价值。
