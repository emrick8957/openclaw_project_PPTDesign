# data_governance 数据治理主题包

## 使用时机

主题涉及数据治理、数据资产、数据中台、数据标准、数据质量、指标体系、数据目录、数据血缘等。

## 推荐结构

治理目标 → 数据现状 → 问题诊断 → 治理框架 → 标准体系 → 质量闭环 → 资产运营 → 场景价值 → 推进路线。

## 推荐表达

- 数据治理的核心不是建目录，而是形成标准统一、质量可控、资产可用、价值可度量的运营体系。
- 应以业务场景牵引数据治理，避免治理与应用脱节。
- 建议先治理高价值数据域，再复制到全域。
