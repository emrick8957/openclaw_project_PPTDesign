# gov_public_security 公安政务主题包

## 使用时机

主题涉及公安、市局、分局、区县、政务、公共安全、视频解析、警情研判、一网统管等。

## 推荐结构

背景与目标 → 业务场景 → 当前问题 → 总体架构 → 能力建设 → 数据/模型/应用协同 → 运营机制 → 风险合规 → 推进计划。

## 推荐表达

- 政务公安场景的核心不是单点能力展示，而是稳定、可控、合规、可运维的持续服务能力。
- 应围绕市局统筹、区县落地、跨域协同构建统一能力底座。
- 建议以高频刚需场景牵引，形成数据、模型、应用、运营的闭环。
