# 参考材料学习流程

## 1. 目标
当用户持续提供 PPT、PPT 图片、模板、文本材料或数据材料时，本流程负责：
1. 对参考材料进行分类；
2. 从视觉与内容两个通道抽取可复用规律；
3. 判断规律应沉淀到哪一层；
4. 控制过拟合、主题污染与事实误用风险；
5. 让后续生成稳定复用已沉淀规则。

## 2. 输入类型分类
常见输入包括：
- 完整 PPT / PPTX；
- PDF 导出的多页截图；
- 单页页面截图；
- 模板页 / 样张页；
- 文本材料；
- 数据表；
- 行业研究或内部分析材料。

## 3. 证据等级
每份材料必须先判断证据等级：
1. 风格参考：用于学习配色、密度、视觉气质；
2. 结构参考：用于学习页面布局、章节结构、图表组织；
3. 论证参考：用于学习标题句式、页型分工、论证套路、决策收口；
4. 事实依据：用于支持具体数据、案例、参数或市场判断。

同一材料可以同时具备多个等级，但事实依据要求最高，不能由截图自动替代。

## 4. 双通道抽取
### 4.1 视觉通道
抽取：
- 配色规律；
- 字体层级；
- 页面密度；
- 页脚规范；
- 常见版式结构；
- 图表主形态；
- 强调信息方式。

### 4.2 内容通道
抽取：
- 章节逻辑；
- 页型角色；
- 标题句式；
- 一页如何形成核心判断；
- 论证链条；
- 决策收口方式；
- 行业框架与方法论套路。

## 5. 沉淀落位规则
抽取结果必须按层级落位：
1. 通用视觉规律 → `templates/`、`visual_patterns/`
2. 通用表达规律 → `templates/wording_rules.md`、`templates/narrative_patterns.md`
3. 通用方法论规律 → `methodology_patterns/`
4. 主题专属框架 → `domain_profiles/`
5. 一次性案例 → 只作为当前任务参考，不进入长期规则库

## 6. 升级条件
只有满足以下条件，才允许进入长期规则：
1. 跨两份及以上材料重复出现；
2. 对多个主题或多个任务有复用价值；
3. 不与现有通用规则冲突；
4. 不依赖单一客户、单一项目、单一行业私有上下文。

## 7. 冲突处理
如果新材料与已有规则冲突：
1. 优先保守，不直接覆盖旧规则；
2. 先判断是否属于主题专属差异；
3. 再判断是否只是单次案例；
4. 必要时将冲突规则降级到主题层或当前任务层。

## 8. 复用优先级
生成时按以下优先级复用：
1. 用户当前明确指定偏好；
2. 已验证的通用规则；
3. 当前主题命中的主题规则；
4. 已验证的方法论规则；
5. 当前任务一次性参考。

## 9. 禁止事项
- 不得把截图中的具体市场结论直接当作事实；
- 不得把单份材料的专有术语升格为通用规则；
- 不得把行业专属章节逻辑写成所有主题的默认结构；
- 不得因参考材料学习而放松主题污染控制。
