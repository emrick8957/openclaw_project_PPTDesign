# 参考材料使用边界

## 1. 目的
定义不同参考材料的证据等级、可用范围与禁止用途，避免把视觉参考误当事实依据。

## 2. 材料类型与默认用途
### 2.1 图片 / 截图
默认可用于：
- 风格参考；
- 结构参考；
- 页面角色识别；
- 图表与版式抽象。

默认不可直接用于：
- 事实断言；
- 精确数据引用；
- 未核验案例引用。

### 2.2 PPT / PPTX 正文
可用于：
- 表达习惯参考；
- 章节结构参考；
- 论证套路参考；
- 页面组织参考。

涉及具体事实时：
- 需要核验其来源是否仍有效；
- 未核验时必须继续标注“待补材料”或“需确认”。

### 2.3 文本材料 / 行业报告
可用于：
- 论证参考；
- 章节结构参考；
- 事实依据候选。

若为权威来源且时效可接受，可作为事实依据。

### 2.4 数据表
可优先作为：
- 图表数据来源；
- 对比和趋势依据；
- 定量支撑。

但仍需检查：
- 口径是否一致；
- 时间是否有效；
- 是否适合直接外显。

## 3. 使用边界
1. 风格参考不等于事实依据；
2. 结构参考不等于通用方法论；
3. 论证参考不等于可直接复述原文；
4. 事实依据必须优先选择可核验来源。

## 4. 输出要求
如果输出中使用了未完全核验的材料，只能做方向性表达，不得冒充确定事实。
